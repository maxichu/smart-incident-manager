 ## F3 — Full-Text Search

 SQLite FTS5 with trigram tokenizer provides ranked full-text search
 across all incident content. Lowercased content ensures case-insensitive
 matching. Snippets are built with <mark> highlighting in Python.

 ### Test Case 1 — Keyword: database
 **Input:** Query = "database"
 **Expected:** Incidents with database content appear, ranked by bm25.

 ### Test Case 2 — Keyword: timeout
 **Input:** Query = "timeout"
 **Expected:** Matching incident appears with highlighted snippet.

 ### Test Case 3 — No Results
 **Input:** Query = "nonexistentkeyword"
 **Expected:** "No matching incidents found" message displayed.

 ### Test Case 4 — Keyword: vpn
 **Input:** Query = "vpn"
 **Expected:** VPN incidents appear with relevant snippets.
