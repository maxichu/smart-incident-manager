 ## F7 — Action Recommendations

 Playbook-based engine maps category + keyword to recommended actions.
 Falls back to category-level actions when no keyword matches, then
 generic fallback when category is unavailable.

 ### Test Case 1 — VPN
 **Input:** VPN unavailable for Singapore users.
 **Expected:** Recommendations include VPN gateway, firewall, restart actions.

 ### Test Case 2 — Database
 **Input:** Database connection timeout during peak traffic.
 **Expected:** Recommendations include DBA escalation and troubleshooting.

 ### Test Case 3 — CPU
 **Input:** Server CPU usage reaches 95%.
 **Expected:** Recommendations include resource investigation actions.

 ### Test Case 4 — Security
 **Input:** Unauthorized access attempt detected.
 **Expected:** Recommendations include security investigation actions.

 ### Test Case 5 — Fallback
 **Input:** Unknown issue with no keyword match.
 **Expected:** Category or generic fallback recommendations returned.
