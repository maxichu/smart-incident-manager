 ## F5 — Incident Classification

 Automatically classifies each incident into one of five IT categories
 using keyword scoring. The category with the most keyword matches wins.
 Falls back to Others when no keywords match.

 ### Test Case 1 — Network
 **Input:** VPN unavailable for Singapore users.
 **Expected:** Category = Network

 ### Test Case 2 — Database
 **Input:** Database connection timeout during peak traffic.
 **Expected:** Category = Database

 ### Test Case 3 — Infrastructure
 **Input:** Server CPU usage reaches 95%.
 **Expected:** Category = Infrastructure

 ### Test Case 4 — Security
 **Input:** Unauthorized access attempt detected.
 **Expected:** Category = Security

 ### Test Case 5 — Application
 **Input:** Frontend deployment causes API errors.
 **Expected:** Category = Application
