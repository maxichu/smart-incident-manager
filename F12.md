 ## F12 — Trend Analysis Dashboard

 Aggregated incident statistics with charts: total count, category and
 severity distributions, and weekly trend line chart.  Powered by SQL
 GROUP BY queries against the existing incidents table.

 ### Test Case 1 — Dashboard Loads
 **Action:** Scroll to Trend Analysis section at page bottom.
 **Expected:** Metrics, bar charts, and line chart render without errors.

 ### Test Case 2 — Category Distribution
 **Action:** View Category Distribution bar chart.
 **Expected:** Bars for Network, Database, Security, Infrastructure,
 Application, and Others with correct counts.

 ### Test Case 3 — Severity Distribution
 **Action:** View Severity Distribution bar chart.
 **Expected:** Bars ordered Critical → High → Medium → Low.

 ### Test Case 4 — Weekly Trend
 **Action:** View Weekly Incident Trend line chart.
 **Expected:** Line chart with week labels on x-axis and counts on y-axis.

 ### Test Case 5 — Sample Data
 **Action:** First run generates ~30 sample incidents across 30 days.
 **Expected:** All categories and severities represented, visible trends.
