 ## F9 — Citation Finder

 Two-stage retrieval: broad candidate fetch (50 most recent) then LLM
 reranking to select top 5 most relevant incident IDs for a question.

 ### Test Case 1 — Affected Clients
 **Q:** How many clients are affected?
 **Expected:** Relevant incident IDs returned.

 ### Test Case 2 — Outage
 **Q:** What caused the outage?
 **Expected:** Outage-related incidents returned.

 ### Test Case 3 — Security
 **Q:** Which incidents were Security related?
 **Expected:** Security category incidents returned.

 ### Test Case 4 — Database Recommendations
 **Q:** What recommendations were given for database failures?
 **Expected:** Database incidents returned with recommendations.
